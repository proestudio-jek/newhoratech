# **App Name**: Semente da Fé

## Core Features:

- Hymn of the Day Display: Display the hymn of the day based on the current date or selected date on the calendar. Hides the hymn title if none is assigned.
- Admin Hymn Management: Enable administrators to add, remove, or update hymns for specific dates on the calendar. This is where a tool would be useful for reasoning.
- Music Player: Stream hymns for each specific hymn.
- Video Gallery: Display music-related videos in a gallery format.
- Image Carousel: Showcase inspirational images in a carousel.
- User Role Management: Restrict the admin hymn management to authorized users.
- Hymn Selection Assistant: AI tool which suggests hymns appropriate for a particular Sunday, feast day, or holiday in the liturgical calendar. Considers hymn text and tune.

## Style Guidelines:

- Primary color: A deep purple (#4B0082) evoking spirituality and reverence.
- Background color: A very light lavender (#F0F8FF) providing a peaceful backdrop.
- Accent color: A warm magenta (#BC13FE) used for interactive elements and highlights, offering a contrast to the primary color.
- Body text: 'Literata', serif, lending a literary and vintage feel.
- Headline font: 'Playfair', sans-serif, lending and elegant and fashionable feel.
- Use simple, elegant icons representing faith, music, and community.
- Implement a clean and intuitive layout with a hamburger menu for navigation.
